BMW Global Sales Analytics Dashboard

An interactive Power BI dashboard for analyzing BMW’s global sales performance, dealer contributions, and market trends — empowering data-driven decisions in the automotive sector.

🏗️ Project Architecture

This project integrates raw sales data with modern analytics tools to generate actionable insights for BMW’s leadership and dealers:

  • 🔄 Data Extraction: Import BMW global sales CSV
  • 🧹 Data Cleaning (Python/Excel): Handle missing records and preprocess data
  • 🗃️ Data Modeling (Power BI): Define relationships between sales, regions, models, and dealers
  • 📊 Visualization (Power BI): Create interactive dashboards for KPIs and comparisons

✨ Key Features

  • 🚗 Model-wise Sales Breakdown: Track revenue, units sold, and growth per model
  • 🌍 Geographic Insights: Map regional and country-level performance
  • 🏬 Dealer Analysis: Identify top-performing dealers and market share
  • 📈 Time-Series Trends: Monthly and YoY performance tracking
  • 🏷️ Profitability Insights: Compare premium vs. standard models
  • ⚡ Interactive Filters: Drilldowns by year, region, or model
